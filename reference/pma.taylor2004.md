# Inositol versus glucose for depressive episode

A pairwise meta-analysis of 4 trials comparing inositol with glucose
(placebo) for depressive episode. The analysed continuous outcome is the
resolution of a depressive episode using the Hamilton Depression Rating
Scale.

## Usage

``` r
data(pma.taylor2004)
```

## Format

A data frame with 4 rows of arm-based data and 11 columns.

## Source

Taylor MJ, Wilder H, Bhagwagar Z, Geddes J. Inositol for depressive
disorders. *Cochrane Database Syst Rev* 2004;**2004**(2):CD004049. doi:
10.1002/14651858.CD004049.pub2

## Details

The interventions have been coded as follows: 1, placebo; 2, inositol.
